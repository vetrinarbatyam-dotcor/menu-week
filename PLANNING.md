## 1. סיכום אפיון (מתוך השאלות)

| נושא | החלטה |
|---|---|
| קהל יעד | אישי קודם, ארכיטקטורה ל-multi-tenant בעתיד |
| משפחה | 2 מבוגרים + 4 ילדים (11, 15, 17, 19) |
| צהריים בבית | 11, 15, 17 |
| טבח ראשי | גיל |
| זמן ערב | 30 דק' |
| בוקר | בבוקר + אופציה ל-prep מערב |
| צהריים | טרי או מחומם |

| דגש תזונתי | חלבון גבוה + דל פחמימות, גמיש לכיוון בריא |
| קניות | פיזי היום → רוצים אונליין; פעמיים בשבוע; גיל+יפצוק |
| שישי | תפריט חגיגי |

| אירוח | פעמיים בחודש |
| פלטפורמה | Web + PWA + בוט Telegram (אופציונלי) |
| משתמשים | גיל+יפצוק עורכים · ילדים צופים+בקשות |
| שפה | עברית בלבד |
| מקור מתכונים | מאגר 200 התחלתי + AI ללמידה והפתעות |

```

│  Frontend (Web + PWA)  │  Telegram Bot (אופציונלי)         │

└──────────┬─────────────┴────────┬────────────────────────────┘
           │                       │
           └───────────┬───────────┘
                       ▼
         ┌─────────────────────────────┐
         │  FastAPI Backend            │
         │  /api/menu  /api/recipes    │
         │  /api/shopping  /api/auth   │
         └──────────────┬──────────────┘
                        │
       ┌────────────────┼─────────────────┐
       ▼                ▼                 ▼
  ┌─────────┐   ┌────────────┐    ┌────────────────┐
  │PostgreSQL│   │ Claude API │    │ Shufersal API/ │
  │ + pgvec  │   │ Gemini API │    │   Scraper      │
  └─────────┘   └────────────┘    └────────────────┘
```

**Multi-tenant ready מיום 1**: כל טבלה עם `family_id` (UUID), אבל עכשיו רק משפחה אחת.

## 3. Stack טכני

| שכבה | טכנולוגיה | למה |
|---|---|---|
| Frontend | React 18 + Vite + TypeScript | מהיר, RTL טוב, PWA out-of-box |
| UI | TailwindCSS + shadcn/ui | מותאם RTL, מודרני |
| Backend | FastAPI (Python 3.11) | תואם שאר הפרויקטים שלך, async |
| DB | PostgreSQL 15 + pgvector | למתכונים + חיפוש סמנטי |
| AI | Claude (סיווג+יצירה) → Gemini fallback | תואם המודל שלך |
| Auth | JWT + bcrypt | פשוט ומספיק |
| Bot | python-telegram-bot | תואם הניסיון שלך |
| Deploy | Docker Compose על Contabo | יחד עם שאר הפרויקטים |
| Port | 3015 (פנוי) | אחרי VetClock 3010-3014 |

## 4. סכמת מסד נתונים (ראשית)

```sql
-- משפחות (מוכן ל-multi-tenant)
families (id, name, settings_json, created_at)
family_members (id, family_id, name, age, role, prefs_json)
  -- role: parent / child / teen
  -- prefs: dislikes[], allergies[], favorites[]

-- מתכונים
recipes (id, name, cuisine, meal_type, prep_minutes,
         protein_g, carbs_g, calories, difficulty,
         ingredients_json, steps_json, tags[],
         source ('seed' / 'ai' / 'user'), embedding vector(1536))
  -- meal_type: breakfast / lunch / dinner / snack / shabbat
  -- difficulty: easy / medium / hard

-- תכנון שבועי
weekly_menus (id, family_id, week_start_date, status, locked_at)
menu_slots (id, weekly_menu_id, day, meal_type, recipe_id,
            servings, notes, is_hosting BOOL)

-- רשימת קניות
shopping_lists (id, weekly_menu_id, generated_at, store, status)
shopping_items (id, list_id, ingredient, quantity, unit,
                category, is_bought, is_in_pantry,
                shufersal_sku, price_cached)

-- למידה
meal_feedback (id, slot_id, member_id, rating 1-5,
               comment, would_repeat BOOL)
pantry (id, family_id, item, quantity, expires_at)
  -- כדי לא לקנות מה שכבר יש
```

## 5. פיצ'רים — MVP מול עתיד

### 🟢 MVP (גרסה 1.0 — שבועיים-שלושה)

1. **תכנון שבוע אוטומטי** — לחיצה אחת, AI בונה תפריט שלם א'-ש'
2. **3 ארוחות/יום** — בוקר, צהריים (לילדים בבית), ערב
3. **התחשבות בזמן** — ערב = 30 דק' מקסימום
4. **שישי חגיגי** — slot מיוחד בעיצוב נפרד
5. **שבת = שאריות + 1 בישול**
6. **מאגר 200 מתכונים** — נטען מראש (ישראלי/איטלקי/אסיאתי/מזיא"ת/מקסיקני/בריא)
7. **דגש תזונתי** — סינון אוטו' לפי חלבון גבוה + דל פחמ'
8. **רשימת קניות אוטומטית** — מכל המתכונים, מקובצת לפי קטגוריה
9. **גרירה ושחרור** — להזיז מתכון בין ימים
10. **"החלף מתכון"** — לא בא לי? בלחיצה AI מציע 3 חלופות
11. **חישוב מנות** — 6 איש כברירת מחדל, אירוח = +X
12. **Web בעברית RTL** — מותאם מובייל ודסקטופ
13. **PWA** — להתקין כאפליקציה במסך הבית

### 🟡 גרסה 1.1 (חודש 2)

14. **בוט Telegram** — "מה לארוחת ערב היום?" / "תוסיף עגבניות לקניות"
15. **תצוגה לילדים** — קישור פתוח ללא רישום, לראות בלבד + כפתור "אני לא רוצה"
16. **בקשות שינוי מילדים** — מסומן להורים לאישור
17. **שמירת העדפות לפי חבר משפחה** — "בן 11 לא אוהב עוף"
18. **מצב prep מערב לפני** — מתכונים שמסמנים "להכין מראש"

### 🟠 גרסה 1.2 (חודש 3)

19. **אינטגרציה Shufersal Online** — סל מוכן בלחיצה (scraping אם אין API)
20. **השוואת מחירים** — שופרסל vs רמי לוי vs יינות ביתן
21. **מצב מזווה** (Pantry) — מה כבר יש בבית, לא לקנות שוב
22. **AI לומד טעמים** — אחרי 4 שבועות, ממליץ אישית
23. **"הפתע אותי"** — תפריט שלם רנדומלי בלחיצה
24. **תוויות תזונתיות** — סה"כ קלוריות/חלבון יומי

### 🔵 גרסה 2.0 (multi-tenant — חודש 6+)

25. **רישום משפחות אחרות** — onboarding flow
26. **תשלום** — Stripe / Tranzila / חודשי
27. **מתכונים שיתופיים** — קהילה
28. **שף אישי** — צ'אט עם AI על אוכל

## 6. UI/UX — מסכים עיקריים

```

├─ "השבוע שלי" — לוח 7×3 (ימים × ארוחות)
├─ כפתור גדול "צור תפריט לשבוע הבא"

⚙️ הגדרות
├─ פרופילי משפחה
├─ העדפות תזונתיות
├─ הפעלת בוט Telegram
```
